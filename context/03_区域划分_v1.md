# 03 区域划分 V1

## 目标

在看到模型结果之前，先固定具有科学意义、可复现、可跨国家对齐的 hydroclimatic regions。

## 区域定义允许使用

```text
lat/lon 或 grid coordinates
ecoregions / bioregions
land cover
cropland fraction / crop frequency
forecast climatology
TB statistics
vegopacity statistics
mask / missingness statistics
elevation / topography if available
```

## 区域定义禁止使用

```text
analysis increments
model errors
target query labels
post-hoc performance
```

## 每个国家的 6 个主 regime slots

```text
R1 dryland / sparse vegetation
R2 semi-arid grassland or agro-pastoral transition
R3 irrigated / intensively managed agriculture
R4 rainfed agriculture
R5 humid high-vegetation
R6 mountain / snow / cold-terrain stress
```

## Region mask 实现

每个区域应保存 mask，而不只是 bbox。

示例 metadata：

```json
{
  "region_id": "US-R1",
  "country": "US",
  "regime": "dryland_sparse_vegetation",
  "bbox_latlon": [32, 37, -116, -109],
  "mask_path": "regions/masks/US_R1.npy",
  "status": "planned",
  "notes": "Use grid audit to map lat/lon bbox to dataset indices."
}
```

## Region quality report

每个 region 必须报告：

```text
grid cell 数量
valid land ratio
obs_mask ratio by year
NaN / Inf counts
time coverage
forecast surface / rootzone SM mean
seasonal amplitude
mean vegopacity
TB-H / TB-V mean and variance
K=4/12/24 support-date availability
```

没有足够 valid support dates 或 query coverage 的区域不能进入正式 split。
