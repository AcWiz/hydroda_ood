# 00 项目总览

## 论文目标

目标会议包括：

```text
ICLR
NeurIPS
ICML
AISTATS
JMLR / Scientific ML
```

CVPR 也可能，但只有在 remote sensing dense prediction / spatiotemporal vision backbone 叙事足够强时才更合适。

## 核心问题

HydroDA-OOD 研究的不是普通土壤湿度预测，而是一个真实部署问题：

> 当 neural land DA operator 被部署到新的水文气候区或新大陆时，如果目标区域只有少量历史 DA analysis cycles，如何进行可靠适配？

## 论文真正的新意

本文的贡献不应只依赖复杂模型，而应来自：

```text
1. Cross-continental OOD benchmark；
2. K-date few-cycle calibration protocol；
3. DA analysis-increment emulation 任务定义；
4. Region-conditioned sparse adaptation；
5. DA-oriented metrics 和 high-update event analysis。
```

## 当前实际约束

当前只有美国数据：

```text
/fastersharefiles2/fenglonghan/dataset/SMAP/DA.nc
```

中国和澳大利亚数据之后会到。因此我们现在先实现：

```text
HydroDA-OOD-US:
  within-US hydroclimatic region OOD
```

之后扩展为：

```text
HydroDA-OOD-Full:
  US / China / Australia cross-continental OOD
```

## 开发原则

不要把 US-only 工作当成临时实验。它应当是完整 HydroDA-OOD 的 development subset。

所有代码抽象都必须支持未来 US/CN/AU。

## 必须保护的论文表述

避免：

```text
We predict true soil moisture.
```

使用：

```text
We emulate reference DA analysis increments.
```

## 第一个成功里程碑

第一个阶段成功的标准是：

```text
给定美国 DA.nc：
  1. 审计 dimensions、time coverage、coords、variables、masks；
  2. 构建 6 个具有科学意义的美国区域 mask；
  3. 生成 K-date support/query splits；
  4. 跑通 Forecast-only、mean increment、monthly increment、ridge baseline；
  5. 输出 6 个 held-out US regions 的 region-balanced metrics。
```
