# 01 研究协议

## 任务定义

对于每个日期、区域和 patch：

```text
输入 x：
  sm_surface_forecast
  sm_rootzone_forecast
  mwrtm_vegopacity
  tb_h_obs
  tb_v_obs
  mask
  time encoding
  optional region descriptor

目标：
  ΔSM_surface  = sm_surface_analysis  - sm_surface_forecast
  ΔSM_rootzone = sm_rootzone_analysis - sm_rootzone_forecast
```

模型可以输出 increment，也可以输出 analysis。但内部训练和评估优先使用 increment。如果模型输出 analysis，需要转换成 increment 后评估。

## 时间协议

```text
Source fit:
  2015-01-01 到 2020-12-31

Source val:
  2021-01-01 到 2021-12-31

Target context:
  2022-01-01 到 2022-12-31

Target query:
  2023-01-01 到 2025-12-31
```

如果实际数据覆盖不同，audit 脚本必须明确报告，并且 split builder 不能静默修改协议。

## K-date 协议

```text
K = 0, 4, 12, 24
```

K 是目标区域可用的 DA analysis dates/cycles 数量，不是 pixels 或 patches 数量。

```text
K=0:
  不使用 target analysis labels。

K=4:
  每个季节/水文季度选 1 个 target support date。

K=12:
  每个月选 1 个 target support date。

K=24:
  每个月选 2 个 target support dates，分别来自上半月和下半月。
```

## Support seeds

至少使用 5 个 seeds，理想使用 10 个 seeds。

同一个 `country, region, K, seed` 下，所有方法必须使用完全相同的 support dates。

## 主实验

### Experiment 1：US-only development OOD

Leave-one-US-region-out：

```text
source = 其他 5 个 US regions, 2015–2020
target context = held-out US region, 2022
target query = held-out US region, 2023–2025
```

这是 development 和 appendix 的重要实验。

### Experiment 2：Leave-one-continent-out

中国和澳大利亚数据到位后：

```text
Train US+CN -> Target AU
Train US+AU -> Target CN
Train CN+AU -> Target US
```

### Experiment 3：Same-regime cross-continent transfer

对每个 regime triad：

```text
train two countries, test held-out country
```

主文优先展示：

```text
Dryland
Irrigated agriculture
Humid high-vegetation
```

附录展示：

```text
Semi-arid transition
Rainfed agriculture
Mountain/cold stress
```

### Experiment 4：Adaptation mechanism ablation

比较：

```text
no adaptation
full fine-tuning
head tuning
bias-only tuning
AdaBN
adapter
LoRA
region latent tuning
sparse block tuning
HyRAO-Meta
```

### Experiment 5：High-update scientific event analysis

预测完成后，再从 query years 中选择事件。事件选择不能用于模型选择。

展示：

```text
true increment
predicted increment
error
sign agreement
forecast baseline
observation / mask context
```

## 信息泄漏规则

禁止：

```text
target query labels 用于 support selection
target query labels 用于 early stopping
target query labels 用于 normalization
target query labels 用于 region selection
analysis increments 用于定义 region regimes
model errors 用于重新定义 regions
```
