# 04 实验矩阵

## Phase A：US-only development

### A1. NetCDF audit

输入：

```text
/fastersharefiles2/fenglonghan/dataset/SMAP/DA.nc
```

输出：

```text
reports/netcdf_audit_us.md
artifacts/netcdf_audit_us.json
```

### A2. US region mask quality

构建 US-R1 到 US-R6 的区域 mask。

输出：

```text
regions/masks/*.npy
reports/region_quality_us.md
artifacts/region_quality_us.csv
```

### A3. US leave-one-region-out

对每个 held-out target region：

```text
source regions = 其他 5 个 US regions
source fit = 2015–2020
source val = 2021
target context = 2022
target query = 2023–2025
K = 0, 4, 12
seeds = 0..4 initially, later 0..9
```

Baseline：

```text
Forecast
Global source mean increment
Target support mean increment
Target monthly support mean increment
Ridge calibration
Small Conv/UNet
Pooled Sformer
```

## Phase B：Full cross-continental benchmark

中国和澳大利亚数据到位后：

```text
Train US+CN -> Target AU
Train US+AU -> Target CN
Train CN+AU -> Target US
```

K 和时间协议保持不变。

## Phase C：Same-regime transfer

对每个 triad：

```text
train two countries, test one country, same regime slot
```

主文：

```text
Dryland
Irrigated agriculture
Humid high-vegetation
```

附录：

```text
Semi-arid transition
Rainfed agriculture
Mountain/cold stress
```

## Phase D：Adaptation ablation

比较：

```text
No adaptation
Full fine-tuning
Head tuning
Bias-only tuning
AdaBN
Adapter
LoRA
Region latent only
Region latent + adapter
Gradient Top-K block tuning
FISA
HISA if feasible
HyRAO-Meta
HyRAO-Meta-FISA/HISA
```

先用美国 hard cases：

```text
US-R1 dryland
US-R3 irrigated agriculture
US-R5 humid high vegetation
US-R6 mountain stress
```

之后扩展到跨大陆 hard cases。

## Phase E：Event-level scientific analysis

只能在 prediction 生成之后，从 query years 中选择事件。不能用事件结果做模型选择。

事件类型：

```text
rainfall pulse / drydown
irrigation or growing-season transition
high vegetation opacity
mountain / cold stress
```

输出：

```text
true increment
predicted increment
forecast-analysis difference
prediction error
sign agreement
obs_mask / TB context
```
