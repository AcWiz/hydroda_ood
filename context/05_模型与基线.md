# 05 模型与基线

## 模型原则

benchmark 和 protocol 是论文骨架。不要在强 baseline 跑通前过度投入复杂方法。

## 必须实现的 baseline

### Forecast-only

```text
ΔSM_hat = 0
SM_hat = forecast
```

### Global source mean increment

从 source fit regions 计算平均 increment，并应用到 target。

### Target support mean increment

K>0 时，从 target support dates 计算平均 increment。

### Target monthly mean increment

K=12/24 时，从 support dates 按月计算 monthly mean increment。这是危险但必须加入的强 seasonal baseline。

### Ridge calibration

输入特征可包括：

```text
forecast surface/rootzone
TB-H
TB-V
TB polarization difference
vegopacity
obs_mask
sin/cos time
optional lat/lon encoding
```

目标是 surface/rootzone increments。

### Small CNN / UNet baseline

在复杂 transformer adaptation 前，先实现透明、稳健的 DA increment neural baseline。

### Collaborator Sformer baseline

合作方 Sformer 可作为强 neural baseline，但必须接入我们的 dataset、split、metric pipeline。

已知兼容条件：

```text
num_vars = 6
patch_size = 4
window_size = 8
原始 full-grid config 的 img_size = [256, 640]
```

对于区域 patch，需要动态适配 img_size 或写 wrapper。

## 主方法路线

### HyRAO-Meta

```text
ΔSM_hat = fθ(x_t, z_r)
z_r = gφ(d_r)
```

其中 `d_r` 是 input-only region descriptor。

K=0：

```text
只用 input-only descriptor 初始化 z_r
```

K>0：

```text
在 target support dates 上适配 z_r 和少量模块
```

### Adapter / LoRA / bias tuning

先作为 baseline 和组件实现。

### Sparse block adaptation

优先做 block-level Top-K，不要一开始做 scalar parameter Top-K。

候选 block：

```text
patch embedding
early spatiotemporal blocks
late spatiotemporal blocks
observation fusion
vegopacity / TB gate
normalization affine params
decoder head
region adapter
```

Gradient score：

```text
score_b = ||g_b||^2
```

Fisher / Hessian-informed score：

```text
score_b = ||g_b||^2 / (curvature_b + lambda)
```

第一版优先实现 FISA。只有在稳定且计算可承受时再实现 HISA。
