## Normalization

输入变量：

```text
x_norm = (x_raw - mean) / std
```

forecast 和 analysis 最好保留 raw values，用 raw values 计算 increment：

```text
increment_raw = analysis_raw - forecast_raw
```

之后可以用 source-train increment statistics 归一化 increment。

严禁使用 target query years 计算 normalization statistics。

## Mask 语义

至少区分三种 mask：

```text
obs_mask:
  输入中的 observation / assimilation availability mask。

region_mask:
  当前 region 对应的空间 mask。

loss_mask:
  训练 loss 和 metric 使用的有效像素 mask。
```

不要把这三者混为一谈。

建议：

```text
loss_mask = region_mask AND finite(forecast) AND finite(analysis) AND optional valid land mask
```

## Padding

不同区域尺寸可以不同。Crop 后 padding 到：

```text
patch_size × window_size = 4 × 8 = 32 的倍数
```

默认：

```python
H_pad = ceil(H / 32) * 32
W_pad = ceil(W / 32) * 32
```

padding 区域：

```text
input = 0 after normalization
obs_mask = 0
region_mask = 0
loss_mask = 0
```

## Patch training

默认训练 patch：

```text
spatial patch = 64 × 64
stride = 32
min_region_valid_ratio = 0.30
```

评估时可以 dense patch reconstruction 或 full-region crop。
