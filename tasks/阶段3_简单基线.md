# 阶段 3：简单 Baseline

## 目标

在 neural modeling 前实现 non-neural 和 linear baselines。

## 必须实现的 baselines

```text
Forecast-only
Global source mean increment
Target support mean increment
Target monthly support mean increment
Ridge calibration
```

## 必须实现的模块

```text
hydroda/models/baseline_linear.py
hydroda/experiments/run_baseline.py
hydroda/metrics/*
```

## 输出

```text
artifacts/predictions/baselines/*.zarr 或 *.npz
artifacts/metrics/baselines_us_loro.csv
reports/baseline_results_us.md
```

## 验收标准

- Forecast-only Skill 精确等于 0。
- Mean / monthly baselines 不访问 target query labels。
- Ridge baseline 可运行 K=4/12/24。
- 结果 region-balanced，并分别报告 surface / rootzone。
