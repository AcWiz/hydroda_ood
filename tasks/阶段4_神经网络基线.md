# 阶段 4：神经网络 Baseline

## 目标

实现透明 neural baselines 和合作方 Sformer wrapper。

## 模型

```text
Small ResUNet 或 ConvNeXt-UNet
Pooled Sformer baseline
Adapter / LoRA-capable wrapper, if feasible
```

## 任务

1. 实现一个小型 neural DA increment model。
2. 在 US source regions 上训练，在 held-out regions 上评估。
3. 只有在 dataset 和 metrics 稳定后，才包装合作方 Sformer。
4. variable count 必须 configurable。
5. region patch / crop size 必须能 padding 到 window/patch requirement。

## 验收标准

- 模型能在一个 LORO split 上训练若干 batches。
- Evaluation 产生全部 metrics。
- 模型输出可解释为 increment。
- 合作方 Sformer baseline 不绕过我们的 split / metric pipeline。
