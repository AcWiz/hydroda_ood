# 待确认问题

## Q1：DA.nc 是否包含 lat/lon coordinates？

需要用于将科学 bbox 映射到 grid masks。

如果没有，需要向合作方索要 projection/grid mapping 或 lat/lon lookup table。

## Q2：`mask` 的确切语义是什么？

可能包括：

```text
observation availability
assimilation mask
valid land mask
target update mask
```

需要 audit 或向合作方确认。

## Q3：time coverage 是否完整覆盖 2015–2025？

如果不完整，需要显式更新 split 协议。

## Q4：中国和澳大利亚 DA.nc 是否使用相同 schema？

需要确认：

```text
variable order
input / target channel counts
grid / projection
normalization
time frequency
mask semantics
```

## Q5：increment 是否单独归一化？

推荐：先 raw increment = analysis - forecast，再用 source-train increment stats 归一化。
需要实验验证稳定性。
